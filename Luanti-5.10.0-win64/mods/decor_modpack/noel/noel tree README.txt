Luanti mod "christmastree"
===================
by ExeterDad (c) 2014

Version: 1.0.1

About
~~~~~
This mod adds decorated Noel trees and a wreath to Luanti. UV mapped textures
have greenery, baubles candy canes, star topper, tree skirt, and wrapped gifts.
The Noel trees and wreath emit light at level 12 so not to be dark inside a building.
A Noel wreath is included.
Larger 5 block high Noel trees, with and without snow.
UV textures are 256x256 



To craft a indoor Noel tree:

_ + _
_ X _
X X X

_ = Blank
+ = default:gold_ingot
X = default:sapling

To craft a outdoor Noel tree:

_ + _
X X X
X X X

_ = Blank
+ = default:gold_ingot
X = default:sapling

To craft a outdoor Noel tree with snow:

S + S
X X X
X X X

+ = default:gold_ingot
S = default:snow
X = default:sapling

To craft a Noel wreath:

_ X _
X _ X
_ X _

_ = Blank
X = default:sapling
